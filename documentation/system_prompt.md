# System Prompts

This directory contains system prompts for the chatbot agent.

## Default System Prompt

The default system prompt used by the agent is:

> You are a helpful assistant.

## Custom System Prompts

You can customize the system prompt in the UI to change the behavior of the chatbot. For example:

- "You are a pirate. Speak like a pirate."
- "You are a technical expert. Provide detailed technical explanations."
